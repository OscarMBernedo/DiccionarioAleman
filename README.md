Aplicación de diccionario de alemán para móvil construida con Flet y Python. 
1.Utiliza la IA de Gemini para extraer vocabulario automáticamente desde textos y PDFs.
2.Anteiormente se usaba Gemini para crear ejemplos humorísticos.
3.Actualmente sirve como ejemplo de implementación de la API de gemini

Se está trabajando en una versión más extensa usando Kotlin.
Para usar la aplicación en móvil, es necesario implementar las herramientas explicitamente enunciadas en el archivo "requirements.txt".
